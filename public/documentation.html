Movie database API
Request URL/Params HTTP Method Request body data format Response body data format
Get list of movies /movies GET none A JSON object holding data about all the movies
Get data about a single movie by name /movies/[name] GET none A JSON object holding data about a single movie.
Get data about a genre /genres/[name] GET none A JSON object holding data about a genre.
Get data about a director /directors/[name] GET none A JSON object holding data about a director.
Create a user account /users POST A JSON object holding data about the user.

Example:
{ "name": "Donnie", "favorites": [] } 		A text message indicating that an account was created.
Update username /users/[id] PUT A JSON object holding data about the user.

Example:
{ "name": "Ronnie", "favorites": [] } 		A text message indicating that the username was changed.
Delete a user account /users/[id] DELETE None A text message indicating that a user account was deleted.
Add a movie to favorites /users/[userId]/favorites/[movieId] PATCH None A text message indicating whether the movie was succesfully added.
Remove movie from favorites /users/[userId]/favorites/[movieId] DELETE None A text message indicating whether the movie was succesfully deleted.
